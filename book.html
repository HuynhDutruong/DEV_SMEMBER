Buổi 1: Giới thiệu HTML và thiết lập môi trường làm việc

1.1 Giới thiệu

Buổi học đầu tiên giới thiệu vai trò của HTML trong phát triển web và hướng dẫn thiết lập môi trường làm việc. Bạn sẽ làm quen với Visual Studio Code (VS Code) và GitHub, hai công cụ thiết yếu để viết và quản lý mã nguồn.

1.2 Lý thuyết

1.2.1 HTML là gì?

HTML (HyperText Markup Language) là ngôn ngữ đánh dấu dùng để tạo cấu trúc cho trang web. HTML sử dụng các thẻ (tags) để định nghĩa các phần tử như tiêu đề, đoạn văn, hình ảnh, liên kết.

1.2.2 Cấu trúc tài liệu HTML

Một tài liệu HTML cơ bản có cấu trúc như sau:

<!DOCTYPE html>
<html lang="vi">
<head>
    <meta charset="UTF-8">
    <title>Trang đầu tiên</title>
</head>
<body>
    <h1>Chào mừng đến với HTML</h1>
    <p>Đây là một đoạn văn bản.</p>
</body>
</html>

- <!DOCTYPE html>: Khai báo đây là tài liệu HTML5.
- <html lang="vi">: Thẻ gốc, chỉ định ngôn ngữ là tiếng Việt.
- <head>: Chứa siêu dữ liệu (metadata) như tiêu đề.
- <body>: Chứa nội dung hiển thị trên trang.

1.2.3 Các thẻ HTML cơ bản

  • <h1> đến <h6>: Tiêu đề từ lớn đến nhỏ.
  • <p>: Đoạn văn.
  • <a href="URL">: Liên kết.
  • <img src="đường_dẫn" alt="mô_tả">: Hình ảnh.
  • <div>: Phân chia khối nội dung.

1.2.4 Thiết lập Visual Studio Code

VS Code là trình soạn thảo mã nguồn mạnh mẽ. Để bắt đầu:

  1. Tải VS Code tại đây.
  2. Cài đặt tiện ích:
    • Live Server: Xem trước trang web trong thời gian thực.
    • Prettier: Tự động định dạng mã nguồn.

1.2.5 GitHub cơ bản

GitHub là nền tảng quản lý mã nguồn. Các bước cơ bản:

  1. Tạo tài khoản tại github.com.
  2. Tạo kho lưu trữ (repository).
  3. Sử dụng lệnh Git để đẩy mã nguồn: 
    git init
git add .
git commit -m "Lần đẩy đầu tiên"
git push origin main

1.3 Thực hành

Thực hiện các bài tập sau trong lớp:

Bài 1: Trang HTML cơ bản

Tạo một trang HTML với:

  • Tiêu đề cấp 1: "Chào mừng đến với trang web của tôi".
  • Một đoạn văn giới thiệu ngắn.
  • Một hình ảnh (sử dụng URL từ internet).

Xem gợi ý

<!DOCTYPE html>
<html lang="vi">
<head>
    <meta charset="UTF-8">
    <title>Trang đầu tiên</title>
</head>
<body>
    <h1>Chào mừng đến với trang web của tôi</h1>
    <p>Xin chào, đây là trang web đầu tiên của tôi!</p>
    <img src="https://via.placeholder.com/150" alt="Hình mẫu">
</body>
</html>

Bài 2: Đẩy mã nguồn lên GitHub

Lưu trang HTML vừa tạo vào file index.html, tạo kho lưu trữ trên GitHub và đẩy mã nguồn lên.

Xem gợi ý

Các bước:

  1. Tạo kho lưu trữ trên GitHub.
  2. Mở terminal trong thư mục dự án.
  3. Chạy: 
    git init
git add index.html
git commit -m "Thêm trang HTML cơ bản"
git remote add origin [URL_kho_lưu_trữ]
git push origin main

1.4 Bài tập về nhà

  • Tạo một trang HTML giới thiệu bản thân với:
    • Tên, tuổi, sở thích.
    • Một hình ảnh đại diện.
    • Liên kết đến một trang web yêu thích.
  • Đẩy mã nguồn lên GitHub và gửi liên kết kho lưu trữ cho giảng viên.

Buổi 2: CSS cơ bản và bố cục trang

2.1 Giới thiệu

Buổi học này giới thiệu CSS, ngôn ngữ định dạng giao diện, giúp trang web trở nên hấp dẫn. Bạn sẽ học cách tích hợp CSS vào HTML và tạo bố cục cơ bản.

2.2 Lý thuyết

2.2.1 CSS là gì?

CSS (Cascading Style Sheets) dùng để định dạng giao diện, kiểm soát màu sắc, kích thước, bố cục của các phần tử HTML.

2.2.2 Cách tích hợp CSS

  • Inline: Trong thuộc tính style của thẻ HTML. 
    <p style="color: blue;">Văn bản</p>
  • Internal: Trong thẻ <style> ở <head>. 
    <style>
    p { color: blue; }
</style>
  • External: Liên kết file .css bằng <link>. 
    <link rel="stylesheet" href="style.css">

2.2.3 Cú pháp CSS

selector {
    property: value;
}

Ví dụ:

p {
    color: blue;
    font-size: 16px;
}

2.2.4 Các thuộc tính CSS phổ biến

  • color: Màu chữ.
  • background-color: Màu nền.
  • font-size, font-family: Kích thước và kiểu chữ.
  • margin, padding: Khoảng cách ngoài và trong.
  • border: Viền.

2.2.5 Box Model

Mỗi phần tử HTML là một hộp gồm:

  • Content: Nội dung (văn bản, hình ảnh).
  • Padding: Khoảng cách trong.
  • Border: Viền bao quanh.
  • Margin: Khoảng cách ngoài.

Ví dụ:

div {
    width: 200px;
    padding: 10px;
    border: 2px solid black;
    margin: 20px;
}

2.2.6 Bố cục cơ bản

  • display: block: Chiếm toàn bộ chiều ngang (ví dụ: <div>).
  • display: inline: Chỉ chiếm chiều ngang cần thiết (ví dụ: <span>).
  • display: inline-block: Kết hợp cả hai.
  • float: left/right: Sắp xếp phần tử sang trái/phải.

2.3 Thực hành

Bài 1: Định dạng văn bản

Tạo một trang HTML với 3 đoạn văn, định dạng bằng CSS external:

  • Đoạn 1: Chữ màu xanh, kích thước 18px.
  • Đoạn 2: Nền vàng, viền đỏ.
  • Đoạn 3: Căn giữa, chữ nghiêng.

Xem gợi ý

File index.html:

<!DOCTYPE html>
<html lang="vi">
<head>
    <meta charset="UTF-8">
    <title>Định dạng CSS</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <p class="para1">Đoạn văn 1</p>
    <p class="para2">Đoạn văn 2</p>
    <p class="para3">Đoạn văn 3</p>
</body>
</html>

File style.css:

.para1 {
    color: blue;
    font-size: 18px;
}
.para2 {
    background-color: yellow;
    border: 2px solid red;
}
.para3 {
    text-align: center;
    font-style: italic;
}

Bài 2: Bố cục hai cột

Tạo một trang HTML với bố cục hai cột (trái: menu, phải: nội dung) sử dụng float.

Xem gợi ý

File index.html:

<!DOCTYPE html>
<html lang="vi">
<head>
    <meta charset="UTF-8">
    <title>Bố cục hai cột</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <div class="menu">Menu</div>
    <div class="content">Nội dung chính</div>
</body>
</html>

File style.css:

.menu {
    float: left;
    width: 30%;
    background-color: #e5e7eb;
    padding: 10px;
}
.content {
    float: right;
    width: 60%;
    padding: 10px;
}

Bài 3: Tùy chỉnh hình ảnh

Thêm một hình ảnh vào trang HTML, định dạng với viền tròn và bóng đổ.

Xem gợi ý

File style.css:

img {
    border-radius: 50%;
    box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2);
}

2.4 Bài tập về nhà

  • Cập nhật trang giới thiệu bản thân từ buổi 1, thêm CSS external với:
    • Màu nền cho trang.
    • Viền và bóng đổ cho hình ảnh.
    • Căn chỉnh nội dung (tiêu đề giữa, đoạn văn thụt đầu dòng).
  • Đẩy mã nguồn lên GitHub.

Buổi 3: TailwindCSS và giao diện responsive

3.1 Giới thiệu

Buổi học này giới thiệu TailwindCSS, một framework CSS giúp tăng tốc thiết kế giao diện. Bạn sẽ học cách tạo giao diện responsive, phù hợp với nhiều thiết bị.

3.2 Lý thuyết

3.2.1 TailwindCSS là gì?

TailwindCSS là framework CSS dựa trên các lớp tiện ích (utility classes), cho phép định dạng giao diện trực tiếp trong HTML mà không cần viết nhiều CSS.

3.2.2 Cài đặt TailwindCSS

Sử dụng CDN để tích hợp nhanh:

<link href="https://cdn.jsdelivr.net/npm/tailwindcss@2.2.19/dist/tailwind.min.css" rel="stylesheet">

3.2.3 Các lớp tiện ích phổ biến

  • bg-: Màu nền (ví dụ: bg-blue-500).
  • text-: Màu chữ, kích thước (ví dụ: text-white text-lg).
  • p-, m-: Padding, margin (ví dụ: p-4 m-2).
  • flex, grid: Bố cục linh hoạt.

3.2.4 Thiết kế responsive

TailwindCSS hỗ trợ breakpoint cho các kích thước màn hình:

  • sm:: ≥576px
  • md:: ≥768px
  • lg:: ≥1024px

Ví dụ: class="text-sm md:text-lg" (chữ nhỏ trên mobile, lớn trên tablet).

3.3 Thực hành

Bài 1: Menu điều hướng

Tạo một menu điều hướng ngang sử dụng TailwindCSS, responsive (ngang trên desktop, dọc trên mobile).

Xem gợi ý

<nav class="bg-blue-600 text-white p-4">
    <ul class="flex flex-col md:flex-row space-y-2 md:space-y-0 md:space-x-4">
        <li><a href="#" class="hover:underline">Trang chủ</a></li>
        <li><a href="#" class="hover:underline">Giới thiệu</a></li>
        <li><a href="#" class="hover:underline">Liên hệ</a></li>
    </ul>
</nav>

Bài 2: Thẻ bài (Card)

Tạo một thẻ bài hiển thị hình ảnh, tiêu đề và mô tả, sử dụng TailwindCSS.

Xem gợi ý

<div class="max-w-sm bg-white shadow-lg rounded-lg p-4">
    <img src="https://via.placeholder.com/300" alt="Hình ảnh" class="w-full rounded-t-lg">
    <h3 class="text-xl font-bold mt-2">Tiêu đề</h3>
    <p class="text-gray-600">Mô tả ngắn về nội dung thẻ bài.</p>
</div>

Bài 3: Bố cục lưới

Tạo một bố cục lưới với 3 thẻ bài trên desktop, 1 thẻ bài trên mobile.

Xem gợi ý

<div class="grid grid-cols-1 md:grid-cols-3 gap-4">
    <div class="bg-white shadow-lg p-4">Thẻ 1</div>
    <div class="bg-white shadow-lg p-4">Thẻ 2</div>
    <div class="bg-white shadow-lg p-4">Thẻ 3</div>
</div>

3.4 Bài tập về nhà

  • Áp dụng TailwindCSS vào trang giới thiệu bản thân:
    • Menu điều hướng responsive.
    • Bố cục lưới cho các sở thích hoặc kỹ năng.
    • Thẻ bài cho thông tin cá nhân.
  • Đảm bảo giao diện hiển thị tốt trên mobile và desktop.
  • Đẩy mã nguồn lên GitHub.

Buổi 4: JavaScript cơ bản và tương tác người dùng

4.1 Giới thiệu

Buổi học này giới thiệu JavaScript, ngôn ngữ lập trình giúp trang web tương tác. Bạn sẽ học cách thao tác với DOM và xử lý sự kiện.

4.2 Lý thuyết

4.2.1 JavaScript là gì?

JavaScript là ngôn ngữ lập trình chạy trên trình duyệt, dùng để thêm tính năng động như nút bấm, biểu mẫu, hiệu ứng.

4.2.2 Tích hợp JavaScript

  • Trong thẻ <script>: 
    <script>
    console.log("Xin chào!");
</script>
  • Liên kết file .js: 
    <script src="script.js"></script>

4.2.3 Cú pháp cơ bản

  • Biến: 
    let ten = "Nguyen";
const tuoi = 20;
  • Hàm: 
    function chao() {
    return "Xin chào!";
}
  • Sự kiện: onclick, onchange, onmouseover.

4.2.4 Thao tác với DOM

  • Chọn phần tử: 
    document.getElementById("id");
document.querySelector(".class");
  • Thay đổi nội dung: 
    element.innerHTML = "Nội dung mới";
  • Thay đổi kiểu dáng: 
    element.style.color = "blue";

4.3 Thực hành

Bài 1: Thay đổi màu nền

Tạo một nút bấm thay đổi màu nền của trang khi nhấp.

Xem gợi ý

File index.html:

<button onclick="doiMau()">Đổi màu</button>
<script>
function doiMau() {
    document.body.style.backgroundColor = "lightblue";
}
</script>

Bài 2: Biểu mẫu đơn giản

Tạo một ô input và nút bấm. Khi nhấp nút, hiển thị giá trị input trong một đoạn văn.

Xem gợi ý

<input type="text" id="nhap">
<button onclick="hienThi()">Hiển thị</button>
<p id="ketQua"></p>
<script>
function hienThi() {
    let giaTri = document.getElementById("nhap").value;
    document.getElementById("ketQua").innerHTML = giaTri;
}
</script>

Bài 3: Hiệu ứng hover

Tạo một div, thay đổi màu nền khi di chuột qua.

Xem gợi ý

<div id="khoi" onmouseover="mauVao()" onmouseout="mauRa()">Di chuột vào đây</div>
<script>
function mauVao() {
    document.getElementById("khoi").style.backgroundColor = "yellow";
}
function mauRa() {
    document.getElementById("khoi").style.backgroundColor = "white";
}
</script>

4.4 Bài tập về nhà

  • Thêm các tính năng tương tác vào trang giới thiệu bản thân:
    • Nút bấm hiển thị/ẩn một phần nội dung.
    • Biểu mẫu để người dùng nhập tên và hiển thị lời chào.
  • Đẩy mã nguồn lên GitHub.

Buổi 5: JSON và quản lý dữ liệu

5.1 Giới thiệu

Buổi học này tập trung vào JSON, định dạng dữ liệu phổ biến trong web. Bạn sẽ học cách tích hợp dữ liệu JSON vào trang web.

5.2 Lý thuyết

5.2.1 JSON là gì?

JSON (JavaScript Object Notation) là định dạng dữ liệu nhẹ, dùng để lưu trữ và trao đổi thông tin.

{
    "ten": "Nguyen Van A",
    "tuoi": 20,
    "kyNang": ["HTML", "CSS"]
}

5.2.2 Làm việc với JSON

  • Đọc JSON: JSON.parse
  • Tạo JSON: JSON.stringify
  • Lấy dữ liệu từ file JSON bằng fetch.

5.2.3 Tích hợp JSON

Sử dụng JavaScript để hiển thị dữ liệu JSON trên giao diện.

5.3 Thực hành

Bài 1: Hiển thị JSON tĩnh

Tạo một đối tượng JSON trong JavaScript và hiển thị dữ liệu lên trang.

Xem gợi ý

<div id="ketQua"></div>
<script>
const duLieu = {
    ten: "Nguyen Van A",
    tuoi: 20
};
document.getElementById("ketQua").innerHTML = `Tên: ${duLieu.ten}, Tuổi: ${duLieu.tuoi}`;
</script>

Bài 2: Đọc file JSON

Tạo file data.json và hiển thị dữ liệu lên trang bằng fetch.

Xem gợi ý

File data.json:

{
    "ten": "Nguyen Van A",
    "kyNang": ["HTML", "CSS"]
}

File index.html:

<ul id="danhSach"></ul>
<script>
fetch("data.json")
    .then(response => response.json())
    .then(data => {
        let ds = document.getElementById("danhSach");
        data.kyNang.forEach(kn => {
            let li = document.createElement("li");
            li.textContent = kn;
            ds.appendChild(li);
        });
    });
</script>

5.4 Bài tập về nhà

  • Tích hợp JSON vào trang giới thiệu bản thân:
    • Tạo file JSON chứa danh sách kỹ năng hoặc sở thích.
    • Hiển thị dữ liệu JSON dưới dạng danh sách hoặc thẻ bài.
  • Đẩy mã nguồn và file JSON lên GitHub.

Buổi 6: Xây dựng bố cục hoàn chỉnh

6.1 Giới thiệu

Buổi học này kết hợp tất cả kiến thức để tạo một bố cục web hoàn chỉnh với header, body, footer.

6.2 Lý thuyết

6.2.1 Cấu trúc giao diện

  • Header: Menu điều hướng, logo.
  • Body: Nội dung chính (bài viết, hình ảnh, biểu mẫu).
  • Footer: Thông tin liên hệ, liên kết.

6.2.2 Tối ưu hóa với TailwindCSS

Sử dụng flex hoặc grid để tạo bố cục linh hoạt.

6.3 Thực hành

Bài 1: Bố cục hoàn chỉnh

Tạo một trang web với header (menu), body (nội dung), footer (liên hệ) sử dụng TailwindCSS.

Xem gợi ý

<header class="bg-blue-600 text-white p-4">
    <nav class="flex justify-between">
        <h1 class="text-2xl">Logo</h1>
        <ul class="flex space-x-4">
            <li><a href="#" class="hover:underline">Trang chủ</a></li>
            <li><a href="#" class="hover:underline">Giới thiệu</a></li>
        </ul>
    </nav>
</header>
<main class="p-4">
    <h2 class="text-2xl">Nội dung chính</h2>
    <p>Đây là nội dung chính của trang web.</p>
</main>
<footer class="bg-gray-800 text-white p-4 text-center">
    <p>Liên hệ: email@example.com</p>
</footer>

Bài 2: Menu điều hướng động

Thêm JavaScript để làm nổi bật mục menu đang được chọn.

Xem gợi ý

<nav class="bg-blue-600 text-white p-4">
    <ul class="flex space-x-4">
        <li><a href="#" class="menu-item">Trang chủ</a></li>
        <li><a href="#" class="menu-item">Giới thiệu</a></li>
    </ul>
</nav>
<script>
document.querySelectorAll(".menu-item").forEach(item => {
    item.addEventListener("click", () => {
        document.querySelectorAll(".menu-item").forEach(i => i.classList.remove("font-bold"));
        item.classList.add("font-bold");
    });
});
</script>

6.4 Bài tập về nhà

  • Bắt đầu dự án cuối khóa: thiết kế giao diện cho cổng thông tin hoặc trang cá nhân với header, body, footer.
  • Đẩy mã nguồn lên GitHub.

7.4 Bài tập về nhà
